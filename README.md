# Fox Adventure

[小狐狸大冒险](https://github.com/hjwforever/FoxAdventure)   

> Unity·2D·横版过关
>
> <kbd>A</kbd>、<kbd>D</kbd>控制小狐狸转向及移动
>
> <kbd>SPACE</kbd> 控制小狐狸跳跃(可二段跳)
>
> <kbd>E</kbd>在适当适合可与场景物品互动，如进入门(第三张图)
